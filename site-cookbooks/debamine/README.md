debamine Cookbook
=================

Description
-----------

Installs [ec2debian-build-ami](https://github.com/andsens/ec2debian-build-ami)
and build Debian EC2 images using it.

Requirements
------------

Debian (squeeze or wheezy) in an AWS EC2 instance.

Attributes
----------


Usage
-----

Include the `default` recipe to the run list.
